# SENG265-Software-Development-Methods

This include basic linux, C, and python usage for small project. Basic debugging and testing with GDB.

The topics from this course include:

- Linux command-line basics
- Multi-version software development
- Software documentation
- C programming language
- Python 3 programming language (including more advanced language use)
- Git version control system
- GDB debugging tool
- Build tools such as "make"
- Testing
- Software evolution and the software life cycle
